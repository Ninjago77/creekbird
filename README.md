\# creekbird

\---

Stealing the faces of all the bros for a competitive slidy bird game!

Link:

\---

\### How to play

UP / W / Space : flap, jump

RIGHT / D / Enter: dash, slide

Enter / Space (GameOver): reset

\---

\### Gamemode Switching

The Game switches from slidy bird to cagey bird around the 35sec mark.

Around the 55sec mark, the game finally switches into super fast flappy bird!

Good Luck
