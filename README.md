# creekbird

Stealing the faces of all the bros for a competitive slidy bird game! Btw: You die when you leave the screen!
Links:
 - Itch: https://ninjago77.itch.io/creekbird
 - Raw: https://creekbird.shantech.pro

## How to play

UP / W / Space : flap, jump

RIGHT / D / Enter: dash, slide

Enter / Space (GameOver): reset

## Gamemode Switching

The Game switches from slidy bird to cagey bird around the 45sec mark.

Around the 150sec mark, the game finally switches into super fast flappy bird!

Good Luck!
